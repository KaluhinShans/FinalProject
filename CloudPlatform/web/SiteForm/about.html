About project

Main idea

Ok, we have a little house, and we need to manage him, so we used rapsberry pi and arduino. When you register at this site and add your devise, you can manage him, see different statistic or turn on(off) light. We create two server(first in rapsberry pi, second in my computer), We create two BD (first is MongoDB in rapsberry pi, second is MySQL in my computer). Arduino get information from sensors, and send their from rapsberry pi, after this first server send information from second server, second server draw info for you. And on the other side.
